Welcome to Alphabet Soup ​

Alphabet Soup is a powerful word search puzzle solver that can find words in any direction - horizontal, vertical, and diagonal (both forward and backward).

Quick Start ​

  1. Load a puzzle - Upload a text file or select an example
  2. View the grid - The word search grid displays on the right
  3. Find words - Click any word in the list to highlight its location
  4. Circle answers - Toggle circle overlays to see all answers at once
