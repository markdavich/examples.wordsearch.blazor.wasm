Architecture

Clean Architecture

Four-layer architecture separating concerns between Domain, Application, Infrastructure, and Web layers.

Blazor WebAssembly

Modern SPA framework running entirely in the browser with C# and .NET.

AI-Powered Parsing

Cloud-based image-to-text extraction using serverless functions and AI vision models.

Component-Based UI

Reusable Blazor components with scoped CSS and event-driven communication.
